# 📋 AI 電子雞 — 需求規格書

> 版本：v1.0 | 日期：2026-03-30 | 狀態：待確認

---

## 1. 概述

AI 電子雞是一款支援 PWA 的網頁遊戲。用戶可以領養虛擬寵物電子雞，透過餵食、互動讓牠成長，若長時間忽視則屬性衰退直至死亡。

---

## 2. 用戶流程

```
註冊帳號
  └─> 為電子雞取名
        └─> 隨機初始化屬性（最多重置 5 次）
              └─> 開始遊戲
                    ├─> 每日餵食（擲骰子機制）
                    ├─> 查看屬性成長
                    └─> 電子雞死亡 → 可領養新的（重置次數恢復）
```

---

## 3. 用戶系統

### 3.1 註冊
- Email + 密碼
- 密碼最少 8 字元
- 註冊後自動導向「命名 & 初始化」頁面

### 3.2 登入
- Email + 密碼
- JWT Token（有效期 7 天）
- 支援「記住我」

### 3.3 電子雞初始化
- 用戶輸入電子雞名字（2-10 字元）
- 點擊「隨機初始化」按鈕，系統在範圍內隨機產生四項屬性
- 顯示目前屬性值與剩餘重置次數
- 確認後開始遊戲，屬性鎖定

---

## 4. 屬性系統

### 4.1 四大屬性

| 屬性 | 初始範圍 | 說明 |
|------|---------|------|
| 生命 ❤️ | 70-80 | 生命值歸 0 則電子雞死亡 |
| 體力 ⚡ | 30-70 | 體力值越大，每日可餵食次數越多 |
| 胃口 🍽️ | 30-70 | 影響每次可餵食的量 |
| 體型 📏 | 20-50 | 數值越大體型越大，吃得越多長得越大 |

所有屬性值範圍：1-100

### 4.2 每日餵食次數計算
```
每日可餵食次數 = floor(體力 / 20) + 1
體力 10-29 → 1 次/日
體力 30-49 → 2 次/日
體力 50-69 → 3 次/日
體力 70-89 → 4 次/日
體力 90-100 → 5 次/日
```

### 4.3 成長階段

| 階段 | 條件 | 圖示 |
|------|------|------|
| 蛋 | 剛領養 | 🥚 |
| 幼兒 | 累計餵食 5 次 | 🐣 |
| 成長 | 累計餵食 15 次 | 🐥 |
| 成熟 | 累計餵食 30 次 | 🐔 |
| 老年 | 累計餵食 50 次 | 👴🐔 |

## 5. 餵食機制（骰子系統）

### 5.1 流程
1. 用戶點擊「餵食」按鈕
2. 畫面播放擲骰子動畫
3. 擲出兩顆骰子（各 1-6），加總（2-12）
4. 根據點數查對照表，觸發對應成長事件
5. 顯示成長結果與屬性變化

### 5.2 骰子點數對照表

| 點數 | 事件名稱 | 效果 |
|------|---------|------|
| 2 | 💥 暴食 | 體型 +5, 胃口 +3, 生命 -2 |
| 3 | 🤢 消化不良 | 生命 -3, 體力 -2 |
| 4 | 🥗 清淡飲食 | 生命 +2, 體力 +3 |
| 5 | 🍎 普通進食 | 生命 +2, 胃口 +1 |
| 6 | 🍱 均衡營養 | 生命 +3, 體力 +2, 體型 +1 |
| 7 | 🍽️ 正常餵食 | 生命 +3, 體力 +1, 胃口 +1, 體型 +1 |
| 8 | 🥩 高蛋白 | 體力 +4, 體型 +2 |
| 9 | 🍰 甜食 | 胃口 +4, 體型 +3, 體力 -1 |
| 10 | 🌟 美食饗宴 | 全屬性 +2 |
| 11 | 💊 營養補充 | 生命 +5, 體力 +3 |
| 12 | 🏆 神級料理 | 全屬性 +4 |

> 屬性最大值 100，最小值 1（生命例外，歸 0 = 死亡）

---

## 6. 衰退機制

### 6.1 衰退時間表
| 距上次餵食 | 效果 |
|-----------|------|
| 2 小時 | 生命 -2, 體力 -1 |
| 4 小時 | 生命 -3, 體力 -2, 胃口 -2 |
| 8 小時 | 生命 -5, 體力 -3, 胃口 -3, 體型 -1 |
| 12 小時 | 生命 -8（額外懲罰） |

### 6.2 衰退頻率
每 30 分鐘後端檢查一次，根據距離上次餵食時間累計計算衰退。

### 6.3 死亡
- 生命歸 0 → 電子雞死亡
- 顯示悼念畫面，可選擇「領養新電子雞」
- 重置次數恢復為 5 次

---

## 7. PWA 功能

- 可安裝至手機桌面
- 推播通知（距上次餵食 1.5 小時後提醒）
- 離線支援（顯示最後一次同步的屬性狀態）

## 8. UI/UX 規格

### 8.1 視覺風格
- 像素風（Pixel Art）
- 主色調：暖色系（米白、橙黃、棕色）
- 字型：像素風字體（Press Start 2P 或類似）

### 8.2 主要頁面
1. **登入 / 註冊頁**
2. **電子雞初始化頁**（命名 + 屬性隨機）
3. **主遊戲頁**（電子雞展示 + 屬性面板 + 餵食按鈕）
4. **餵食動畫頁**（骰子動畫 + 結果展示）
5. **死亡頁**（悼念 + 重新領養）
